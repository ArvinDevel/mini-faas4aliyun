# [实现一个 Serverless 计算服务调度系统](https://tianchi.aliyun.com/competition/entrance/231793/introduction)
初次使用golang实战，感觉golang还是挺有意思的语言，最后勉强留在了榜单第一页。
golang比较灵活，相比Java而言，我觉得容易散，代码量容易膨胀，容易脱离体系。
所以写golang更得控制住自己的手，注意代码设计、层次划分。
